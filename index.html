Overview

Forkserf is an open-source improved-clone of the original Settlers (EU) / Serf City (US) game by Blue Byte software. Forkserf was forked from FreeSerf by JonLS and wicked_digger. It is written in C/C++ and published under GPL3+ license.

Why Forkserf?

Forkserf was created to improve the "single-player versus AI" gameplay of the original game. This is done by completely rewriting the original AI from scratch, fixing bugs, and addition of optional features to make the game more balanced and interesting. Also, as far as I know, FreeSerf is the only implementation that allows playing both DOS and Amiga versions in one binary.

Why *not* Forksef

If you are looking for an authentic reproduction of the original game AI, single-player Missions, and multiplayer(!), I suggest you try nicymike's Serflings Or, buy the original game or find on abandonware and run in DosBox, it works perfectly. I don't know how to run the orginal Amiga version, though. Or, try one of the official (UbiSoft?) remakes though I have read they are worse than the original game because they introduce bugs, limit features, change some game logic? not sure

Requires original game content files

For copyright reasons, the original DOS and Amiga game data cannot be included in this distribution and so you must provide your own copy. These can be found in the original game install directories, and on various abandonware sites.

Improvements over FreeSerf & original Serf City / Settlers game

  • advanced AI for enemy players (FreeSerf has no AI at all!
  • fixes for numerous bugs, including game-breaking bugs
  • custom Map Generator
  • more debugging tools, including overlays for highlighting AI logic

Known Limitations

  • single player Missions from original game are included by not playtested, I only play random games
  • MacOS builds provided but not playtested, I don't have a Mac
  • Windows8+ builds provided but not playtested, I only have Windows7
  • No multiplayer, and no immediate plans for multiplayer
  • the original game Character personalities are not yet implemented, all AI players use same improved AI. Personalities will be added soon
  • combat playtesting and AI combat logic is primitive. It works fine but I haven't paid as much attention to it as the economy part
